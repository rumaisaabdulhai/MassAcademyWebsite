Image Classification Project



I have been fascinated with how image classification works in the field of computer vision as it has various applications in medicine, security, and self-driving cars. For example, doctors use image classification in CT scans to identify cancers and tumors. Law enforcement can use image classification to identify suspects. The most interesting application, however, is in autonomous vehicles such as self-driving cars, where pedestrians, objects, or animals must be identified quickly in time for the car to perform obstacle avoidance. My goal is that if an unclassified image is fed into my program, it will identify the object by classifying it into one of several categories.

Apps For Good Project



During D Term, I developed an Android Studio Application in Java to help high school students search for volunteer opportunities that interest them. The first main feature of the app is that students may sort opportunities by location and by name. The second is that they can save opportunities to view later by marking it as favorite. First, I used Web Scraping to obtain the title, description, location, contact information, and more for about 135 volunteer opportunities and created a custom database. I stored the database on Firebase, a service by Google for storing data on the cloud. The Android Application reads and displays the data from Firebase. You can view the Java code here, and the Web Scraping code here.



The left video shows a demo of the application. When the user opens the app for the first time, they are brought to a welcome screen that walks them through the app's features. They are then prompted to create an account or sign in. Any opportunities that they favorite or unfavorite will be updated and saved on the cloud and accessible from any device as long as they can sign in.
