RRT Path Planning Implementation

Path planning algorithms are neccessary to determine an optimal path from a chosen source point to a destination point and are used extensively in the fields of robotics. The Rapidly Exploring Random Tree (RRT) Algorithm caught my eye as it can determine a possible path in a very short amount of time. I used path planning after I mapped the indoor environment so my drone could navigate from one location to another. I coded the RRT Algorithm in a simple 2D Python Simulator called Pygame.
